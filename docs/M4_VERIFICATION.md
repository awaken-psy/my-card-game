## 测试说明

- **启动方式**：在 Godot 编辑器中按 F5 运行
- **测试卡组**：5× Strike + 4× Defend + 1× Bash = 10 张
- **M4 核心验证项**：1-5, 11-12（使用起手牌组即可完成）
- **可选验证项**：6-10（需修改 load_starting_deck 加入奖励牌池卡牌）
- **延后验证项**：10（虚弱减伤，需 M5 敌人系统）

### 已知限制
- 敌人不还手（M5 范围），所以 Block 在 M4 中没有实际减伤效果，只能验证 UI 显示
- 敌人 HP 归零后无胜利画面（M7 范围）
- 状态效果（易伤/虚弱/力量）的 tick-down 目前只在玩家开始回合时触发，M5 敌人回合时需补充敌人的 tick
