## 2. 敌人回合执行 — Chomp（11 伤害）

**操作**：不出任何牌，直接点击 **[End Turn]**。

| 检查项 | 预期表现 | ✅ |
|--------|----------|----|
| 手牌丢弃 | 所有手牌飞到弃牌堆 | [x] |
| 等待 ~0.8s | 意图仍然可见（展示期间） | [x] |
| 玩家受伤 | `❤️ 80/80` → `❤️ 69/80`（-11） | [x] |
| 意图消失 | 敌人意图标签变空（执行后清除） | [x] |
| 短暂停顿 | ~0.5s 后开始新回合 | [x] |

**伤害公式**：
```
enemy_damage = base + enemy.strength
if enemy.weak > 0:     damage = int(damage × 0.75)
if player.vulnerable > 0:  damage = int(damage × 1.5)
damage = max(0, damage)
```

- [x] Chomp 造成 11 点伤害（80 → 69）
- [x] 意图在执行后正确清除

## 6. 敌人格挡减伤

**操作**：等敌人使用了 Thrash（🛡️5）或 Bellow（🛡️6）后，下一回合打出 Strike（6 伤害）。

| 检查项 | 预期表现 | ✅ |
|--------|----------|----|
| 敌人 HP 减少少于 6 | Strike 6 dmg 被格挡吸收部分 | [x] |
| 敌人格挡减少 | `🛡️5` → `🛡️0`（6-5=1 实际 HP 损伤） | [x] |

> 示例：敌人有 🛡️5，Strike 造成 6 伤害 → 格挡吸收 5，HP 只减 1。

- [x] 敌人格挡正确吸收伤害
- [x] 剩余伤害扣除 HP

## 7. 敌人回合格挡清零 + 状态 tick

**操作**：对敌人使用 Bash（施加 2 层易伤 🔻2），然后经过 1 个敌人回合。

| 检查项 | 预期表现 | ✅ |
|--------|----------|----|
| Bash 后敌人显示 `🔻2` | 易伤 2 层正确施加 | [x] |
| End Turn 后敌人格挡清零 | 上回合敌人获得的 🛡️ 消失 | [x] |
| 敌人状态 tick | `🔻2` → `🔻1`（每个敌人回合 -1） | [x] |
| 再过一回合 | `🔻1` → 消失（tick 为 0 时不显示） | [x] |

**敌人回合开始时**：
1. `enemy.reset_block()` — 格挡归零
2. `enemy.tick_status()` — 易伤/虚弱各 -1

- [x] 敌人格挡在敌人回合开始时正确清零
- [x] 敌人易伤每回合递减 1
- [x] 易伤归零后不再显示
